Ferramenta exclusiva para gerar códigos de ativação Premium

⚠️ MANTENHA ESTE ARQUIVO EM SEGREDO. Qualquer pessoa com acesso a ele pode gerar códigos válidos. Não compartilhe e não publique online.
Como usar:
1. Defina o intervalo de índices (ex: 1 a 20 para os primeiros 20 clientes)
2. Clique em Gerar Códigos
3. Copie um código por cliente e anote na sua planilha
4. Envie o código pelo WhatsApp após confirmar o pagamento PIX

Importante: Cada índice gera sempre o mesmo código. Use índices diferentes para clientes diferentes. O app aceita índices de 1 a 500.
